📄 Online Resume Submission Portal

A simple and user-friendly web application that allows users to submit their name and upload their resume through an online form. This project is built using HTML, CSS, and Flask.

🚀 Features
🧑‍💼 User can enter their name
📄 Resume upload functionality
💾 Stores uploaded resumes on the server
🌐 Simple and clean user interface
⚡ Fast and lightweight application

🛠️ Tech Stack
Frontend: HTML, CSS
Backend: Flask (Python)
File Handling: Python (Werkzeug)
